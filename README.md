# Inventory Management System (C)

## Overview

Inventory Management System is a menu-driven application developed in C programming language. The project helps manage product records, inventory tracking, stock monitoring, and user authentication using file handling and structures.

## Features

- User Authentication System
- Add Product Records
- View Product Records
- Search Product Details
- Update Product Information
- Delete Product Records
- Low Stock Alerts
- Inventory Tracking
- File Handling for Data Storage

## Technologies Used

- C Programming
- Structures
- Functions
- File Handling
